# ふたりの

基于 `https://harumakigohan.com/ftr/` 参考实现的静态还原项目，当前包含首页首屏轮播、专辑信息区、3D 展示区、曲目区、角色弹窗与试读翻页区。

## 项目目标

- 复刻原站的视觉风格与主要交互节奏
- 以原生 `HTML + CSS + JavaScript` 维护，避免额外构建依赖
- 将素材统一存放在 `assets/`，方便替换与继续微调

## 目录结构

```text
.
├─ assets/        页面图片、GIF、图标与背景素材
├─ index.html     页面结构
├─ styles.css     全站样式
├─ main.js        首页轮播、入场动画、弹窗与翻页逻辑
└─ README.md      项目说明
```

## 本地运行

直接双击 `index.html` 可预览，但建议使用本地静态服务器以避免浏览器对资源与脚本的限制。

```bash
python -m http.server 8000
```

启动后访问：`http://127.0.0.1:8000/`

### 本次修复内容

本次针对“首页轮播图错位”做了结构与脚本层面的修复：

- 移除了写死在 `HTML` 中的前后克隆节点，避免轮播节点数与样式宽度公式不一致
- 保留 `4` 张真实轮播图，由 `JavaScript` 在运行时动态补 `1` 张克隆图
- 让轮播始终按 `5` 槽位模型移动，和原站的位移计算方式保持一致
- 为轮播容器增加 `white-space: nowrap`、`font-size: 0` 和显式居中，减少 inline-block 带来的间隙与错位
- 在窗口尺寸变化时重置轮播偏移，避免缩放后首屏封面位置漂移
